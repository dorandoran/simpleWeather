## Testing

#### Prerequisites
[OpenWeatherMap API Key](https://openweathermap.org/api)

#### Installing
Clone the repository in your terminal:
```sh
 git clone https://github.com/dorandoran/simpleWeather
```
**Make sure you insert your API Key!**

> In app.js, line 13:
```javascript
  var apiKey = '[INSERT API KEY]';
```
